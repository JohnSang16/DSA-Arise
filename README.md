## ⚔️ DSA Quest

<strong>An immersive RPG experience that turns Data Structures and Algorithms (DSA) into a legendary adventure.</strong>

## 📖 Overview
<strong>DSA Quest</strong> is a progression-based learning platform where coding challenges are reimagined as epic battles. You play as Arthur, starting as a simple Initiate in the Null Plains. By solving DSA problems, you gain health, upgrade your armor, and eventually ascend to the rank of Paladin. The curriculum is specifically mapped to the <strong>GSU-DSA</strong> standard to provide a practical yet addictive way to master computer science fundamentals.


## 📦 Technologies
<ul>
  <li><strong>Frontend</strong>: React / Next.js with Framer Motion and Tailwind CSS.</li>
  <li><strong>Backend</strong>: Python 3 with Flask Framework</li>
  <li><strong>Game Engine:</strong>Phaser.js handles the 2D sprites, collisions, and animations.</li>
  <li><strong>Code Editor</strong>: Monaco Editor for a professional-grade IDE experience. </li>
</ul>



## ✨ Features
<ul>
  <li><strong>Hero Progression:</strong> Arthur begins at 50/100 HP. Every level cleared grants +10 HP recovery and visual armor upgrades (Initiate → Guardian → Veteran → Elite → Paladin → Legendary).
</li>
  <li><strong>Narrative Problem Solving:</strong> Instead of dry inputs/outputs, you arrange chest items to open doors or filter magical projectiles to block critical hits.
</li>
  <li><strong>Visual Feedback: </strong> Every function call (e.g., list.pop()) triggers a physical animation where items leave the screen or pointers lock onto new nodes.
)</li>
  <li><strong>Boss Battles:</strong>  End-of-stage challenges (like the Null Sentinel) require multi-target logic and advanced algorithmic efficiency.
</li>
</ul>



## 💡 Project Motivation
Technical interview prep and DSA coursework are often met with fatigue. Inspired by the narrative flow of Flexbox Adventure and the progression loops of modern RPGs, I built this to bridge the gap between theoretical study and active, visual engagement. This project is my sandbox for combining System-Level Programming with interactive web development.
