# ⚔️ DSA Arise

<strong>A dark anime RPG experience that turns Data Structures and Algorithms into a legendary ascent.</strong>

---

## 📖 Overview

<strong>DSA Arise</strong> is a Solo Leveling-inspired progression platform where coding challenges are reimagined as dungeon battles. You play as <strong>Kai</strong>, starting as a powerless <strong>E-Rank Hunter</strong> in the depths of a corrupted data dungeon. A mysterious System awakens within him, and from that moment, everything changes.

By solving DSA problems, Kai defeats corrupted data entities, earns XP, and ascends through the hunter ranks. The curriculum is specifically mapped to the <strong>GSU-DSA</strong> standard to provide a practical yet addictive way to master computer science fundamentals.

| Rank | Stage | DSA Topic |
|------|-------|-----------|
| E-Rank | Stage 1 | Arrays |
| D-Rank | Stage 2 | Stacks & Queues |
| C-Rank | Stage 3 | Linked Lists |
| B-Rank | Stage 4 | Trees / BST |
| A-Rank | Stage 5 | Graphs |
| S-Rank | Cleared | All stages complete |

## 📦 Technologies

<ul>
  <li><strong>Frontend:</strong> React + Vite with Framer Motion for cinematic animations</li>
  <li><strong>Routing:</strong> React Router DOM</li>
  <li><strong>Auth + Database:</strong> Supabase (auth, progress tracking, rank storage)</li>
  <li><strong>Code Editor:</strong> Monaco Editor for a professional grade IDE experience</li>
  <li><strong>Styling:</strong> CSS Modules with a dark anime system window aesthetic</li>
  <li><strong>Fonts:</strong> Press Start 2P + Rajdhani</li>
</ul>

---

## ✨ Features

<ul>
  <li><strong>Hunter Rank Progression:</strong> Kai begins at E-Rank with 50/100 HP. Every level cleared grants +10 HP and advances his rank; visually reflected in his sprite, eye glow intensity, mana aura, and UI rank badge.</li>
  <li><strong>Solo Leveling System Windows:</strong> Instead of plain UI, every quest, rank-up notice, and system message appears as a dramatic dark navy system window with electric blue borders — straight out of the dungeon.</li>
  <li><strong>Cinematic Opening Cutscene:</strong> An animated intro sequence tells Kai's origin story, from flagged E-Rank candidate to chosen player — before a single line of code is written.</li>
  <li><strong>Narrative Problem Solving:</strong> Every DSA challenge is framed as a dungeon encounter. Defeat the DataSlime by mastering array indexing. Destroy the HydraNode by deleting the correct linked list node.</li>
  <li><strong>Visual Feedback:</strong> Every correct answer triggers an attack animation. Wrong answers shake the panel. Rank-ups trigger a full system window sequence with mana burst effects.</li>
  <li><strong>Boss Battles:</strong> End-of-stage level 5 challenges require full code solutions, not just fill-in-the-blank. Beat the boss, clear the dungeon, rise in rank.</li>
</ul>

## 💡 Project Motivation

Technical interview prep and DSA coursework are often met with fatigue. Inspired by the narrative progression of Solo Leveling, the split-pane UI of Flexbox Adventure, and the progression loops of modern RPGs, I built DSA Arise to bridge the gap between theoretical study and active, visual engagement. This project is my sandbox for combining system level programming with interactive web development, and proving that learning DSA can feel like an ascent, not a grind.
